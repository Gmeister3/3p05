# 3p05

## COSC 3P03 Assignment 1 Solution

This repository contains the LaTeX solution for Assignment 1 of COSC 3P03 (Winter 2026).

### Files
- `assignment1_solution.tex` - LaTeX source file with complete solutions
- `assignment1_solution.pdf` - Compiled PDF with all answers
- `Assignment1 (1).pdf` - Original assignment problem statement

### Topics Covered
1. Narayana Cow Sequence (recursion trees, solving recursive equations, proof by induction)
2. Lucas Numbers (bonus question)
3. Minheap Data Structure (array and linked list implementation, extractMin operation)
4. Asymptotic Notations (Big-O, little-omega analysis)
5. Bubble Sort Analysis (time complexity, proof of correctness)
6. Celebrity Problem (implementation note - code to be submitted separately)

### Compiling the LaTeX
To compile the LaTeX document:
```bash
pdflatex assignment1_solution.tex
```
